# Fonts Directory

Place your TrueType font files (.ttf) here.

## Recommended Fonts

- **Arial-Bold.ttf** - For titles and headings
- **Arial.ttf** - For body text and content

## Font Usage

The system is configured to use:
- Bold fonts for titles and CTAs
- Regular fonts for content lines and brand names

If custom fonts are not provided, the system will fall back to system default fonts.

## Where to Get Fonts

- [Google Fonts](https://fonts.google.com/)
- System fonts (macOS: `/System/Library/Fonts/`, Windows: `C:\Windows\Fonts\`)
