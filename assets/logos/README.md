# Logos Directory

Place your brand logo files here.

## Format

- **Recommended**: PNG with transparent background
- **Size**: At least 500x500 pixels for best quality

## Naming Convention

Logo files should match the brand configuration:

- `gym_college_logo.png` - The Gym College brand
- `fitness_pro_logo.png` - Fitness Pro brand
- `wellness_life_logo.png` - Wellness Life brand

## Usage

Logos will be displayed at the bottom of reel images. Ensure:

- High resolution for clarity
- Transparent background for clean integration
- Appropriate contrast with brand colors

## Creating Logos

If you don't have logo files yet, you can:

1. Use a design tool (Canva, Figma, Adobe Illustrator)
2. Hire a designer
3. The system will work without logos (text-only branding)
